# Scripts for determination of nitrate in xylem sap by CE
## 01-input_data
- Contains raw data for calibration, LOD/LOQ, percent recovery, and sample data
## 02-calibration_curve
- Notebook for calculating and evaluating calibration curve data
- Calculates LOD/LOQ and percent recoveries
- Contains script for rigorous error propagation
## 03-statistical_analysis
- Calculates nitrate concentrations from raw data
- Performs statistical analyses on raw data
 - Plots log-transformed data with linear regression model
 - Performs tests for normality and heteroscedasticity in dataset
