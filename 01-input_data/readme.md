# Input data
- `cal_curve.csv` - raw data from calibration curve standards
- `fruit_set.csv` - sample data from xylem sap droplets collected at fruit set
- `lfb_data.csv` - Low and mid LFB data collected on each day of analysis
- `lod_data.csv` - Data for calcualting LOD and LOQ
- `nitrite_nitrate_TFA.CSV` - Exported raw data from Agilent MassHunter for plotting example electropherogram of standards
- `V-15N-5-r001.CSV` - Exported raw data from a xylem sap sample
